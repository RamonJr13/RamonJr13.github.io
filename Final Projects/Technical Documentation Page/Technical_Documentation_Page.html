Introduction to HTML and CSS

HTML (HTMLText Markup Language):

  • HTML stand for HyperText Markup Language.
  • It's the foundation for creating web pages.
  • HTML uses tags to define different parts of webpage, like heading , paragraphs, images, and links.

CSS (Cascading Style Sheets):

  • CSS is a language we use to style a Web page.
  • CSS described how HTML elemants are to be displayed on screen, paper, or in other media.

Git

- Git is an Open Source Distributed Version Control System.

Git Commands

Here are some common Git commands:

  • git clone (repository_url) - Clone a repository that already exist on Github, including all of the files, branches, and commits. This is only done once, when you begin working on a project.
  • git add (file) - Adds a specific file to the staging area.
  • git add . - Adds all of the files in the current directory.
  • git commit -m "Initial commit" - Creates new commit with the changes in the staging area and specifies the commit message inline.
  • git push - uploads all local branch commits to the corresponding remote branch.
  • git pull - updates your current local working branches, and all of the remote tracking branches.
Windows Subsystem for linux (WSL)

Windows Subsystem for linux (WSL) is a feature of windows that allows you to run a linux environment on your windows machine, without the need for a separate virtual machine or dula booting.

WSL is designed to provide a seamless and productive experience for developers who want to use both windows and linux at the same time.

WSL Commands

List of Commands:

  • wsl --install - install a tool to run linux on your computer.
  • wsl - Opens up linux on your windows computer.
  • logout or exit - Closes the linux session and go back to windows, you can type either 'logout' or 'exit'.
  • clear - Clear the terminal screen, removing all you typed.
